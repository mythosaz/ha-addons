# Changelog

All notable changes to this add-on will be documented in this file.

## [1.0.0] - 2026-01-05

### Added
- Initial release
- Forked from openai_image add-on
- Long-running service (no timeout limits)
- Support for OpenAI GPT-image and DALL-E models
- Automatic file versioning (timestamped archives + current version)
- Home Assistant event firing on completion
- Configurable defaults (model, quality, size, output directory)
- Prompt input via direct string or file path
- stdin-based command interface
